# Responsive Landing Page Islands

Responsive Landing Page Islands — a clean, modern, responsive landing page built with plain HTML, CSS, and JavaScript.

![Desktop view](./Screenshot%202024-06-02%20190916.png)

A lightweight template showcasing "islands"-style sections that rearrange across breakpoints for readable, mobile-first layouts.

## Features

- Responsive, mobile-first design using CSS Grid and Flexbox
- Minimal, dependency-free HTML/CSS/JS
- Reusable content "islands" for modular sections
- Simple animations and smooth interactions
- Ready to fork and customize for your projects

## Getting started

To run the project locally:

1. Clone the repository:

```bash
git clone https://github.com/BinaryVortex/Responsive-Landing-Page-Islands.git
cd Responsive-Landing-Page-Islands
```

2. Open `index.html` in your browser, or use a simple static server:

```bash
# Python 3
python3 -m http.server 8000
# then open http://localhost:8000
```
